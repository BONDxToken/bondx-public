Local dev server (optional, faster reloads)
-------------------------------------------
You can open index.html directly, or run a quick local server so browser caching behaves better:

Python (built-in):
  cd bondx-mock-v4
  python -m http.server 8000
Then visit http://localhost:8000/

Node (if you have it):
  npx serve .
